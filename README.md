# LDTclothing

Site vitrine e-commerce inspire d'un style premium/minimal.

## Noms de fichiers simplifies

- Dossier ambiance: `assets/ui/`
  - `hero-main.jpg`
  - `hero-reel.mp4`
  - `universe-01.jpg`
  - `universe-02.jpg`
- Dossier produits: `assets/products/`
  - `product-01.jpg` a `product-11.jpg`

Tu peux modifier les noms/prix/taille directement dans `script.js`.

## Ouvrir le site

Ouvre simplement `index.html` dans ton navigateur.
